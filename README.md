# Library_Management_System

The Library Management System is a full-stack Java Spring Boot application designed to manage libraries, books, and users efficiently. It allows libraries to maintain their book collections, track borrowed/returned books, and manage user interactions in a digital way. 

FEATURES: 

        1. Library Management Add new libraries with details such as name, phone, and address. View and update library details. Associate multiple books with a single library.
        2. Book Management Add new books with title and author. Assign books to a library. Track borrowed books with borrowedTime and returnTime. Mark books as borrowed or returned.
        3. User Management Add users with details (name, phone, email, , address). Allow users to borrow and return books. Track which user has borrowed which book.
        4. Address Management Store addresses for both users    and libraries. Includes house number, area, city, state, country, and pincode.
